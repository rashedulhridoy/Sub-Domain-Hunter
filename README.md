How To Use This Subdomain Hunter ?

First open Your Linux Terminal, Then Type

$ pip install requests

$ pkg install git

$ git clone https://github.com/rashedulhridoy/Subdomain-Hunter.git

Then The Tools is Automatically Download

Then Type

$ cd Subdomain-Hunter

$ python sdh.py
